# DecodeLabs Frontend Internship — Project 1

## Static Webpage Design

This project was developed during my Frontend Development Internship at DecodeLabs as part of the Industrial Training Program 2026.

The main objective of this project was to build a clean and responsive static webpage using only HTML and CSS while following proper frontend development practices.

---

## 📌 Project Overview

This project focuses on:

- Proper HTML page structure
- Semantic sections and headings
- Clean webpage layout
- CSS styling fundamentals
- Responsive and readable design
- Basic UI structuring techniques

The project helped strengthen my understanding of frontend development fundamentals and improved my ability to convert webpage ideas into real user interfaces.

---

## 🚀 Technologies Used

- HTML5
- CSS3
- Google Fonts
- Flexbox
- Responsive Design Principles

---

## 🎯 Features

- Responsive webpage layout
- Structured navigation section
- Hero section
- Services section
- About section
- Contact/Footer section
- Modern typography and spacing
- Clean and maintainable code

---

## 📚 Skills Learned

Through this project, I practiced and improved:

- Semantic HTML structure
- CSS styling techniques
- Layout management
- Spacing and alignment
- Responsive design basics
- UI design implementation

---

## 🛠️ Project Goal

The purpose of this internship task was to understand how frontend developers create structured static webpages before moving toward advanced dynamic web applications.
